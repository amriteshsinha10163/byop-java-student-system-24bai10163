# Java BYOP Project

## Project Title
Student Management System

## Description
This project is a simple console-based application developed in Java to manage student records.

## Features
- Add student
- View students
- Delete student

## Technologies Used
- Java
- OOP concepts
- ArrayList

## How to Run
1. Install Java
2. Compile:
   javac Main.java
3. Run:
   java Main
